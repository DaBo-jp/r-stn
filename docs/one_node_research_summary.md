# **R-STN 基礎理論 検証報告書：フェーズ1 (Unit Dynamics)**

**作成日:** 2025年12月27日

**対象バージョン:** Ver 1.10 (Weightless / Center-Localized Dendritic Growth)

**検証ステータス:** **PASSED**

## **1. 検証目的**

単一ノードにおける「重みなき物理演算」の妥当性を、非線形な環境変化、バーストノイズ、および代謝サイクル（生存・死滅・再生）の観点から実証する。

## **2. 物理パラメータ設定 (ゲル状高耐性モデル)**

本検証では、ノイズ耐性と適応性のバランスを最適化した「ゲル状」の物理特性を採用した。

| パラメータ | 設定値 | 物理的意義 |
| --- | --- | --- |
| **慣性 ()** | 0.99 | 質量。過去の運動状態を強く維持し、一過性の外力を無視する。 |
| **粘性 ()** | 0.35 | 抵抗。急激な加速を抑制し、滑らかな追従性を確保する。 |
| **不感帯 ()** | 1.0 Hz | ヒステリシス。1.0Hz未満の微細な揺らぎを「握りつぶす」。 |
| **寿命幅** | 500 - 1100 | 代謝周期。個体ごとの多様性によるリソース循環の担保。 |

## **3. 検証フェーズ別 観測結果**

### **3.1. 非線形追従 (Phase 4)**

* **観測:** 正弦波ターゲットに対し、慣性によるわずかな位相遅延を伴いつつ、極めて滑らかに追従した。
* **評価:** 0.99の慣性が「重み」の代わりとして機能し、不安定な発振を抑えつつ意図を捉え続けている。

### **3.2. バーストノイズ耐性 (Phase 5)**

* **観測:** 強烈なノイズ（±25Hz）重畳下でも、共鳴振幅の瞬時的な脱落はあるものの、自己周波数の中心値はターゲットから逸脱しなかった。
* **評価:** 慣性と不感帯の相乗効果により、高周波ノイズが物理的にフィルタリングされている。

### **3.3. 断続信号と代謝回復 (Phase 6)**

* **観測:** 信号遮断（休息）時に疲労値（Fatigue）が速やかに減少する「鋸歯状の推移」を確認。
* **評価:** 休息を挟むことで生存期間が延長され、過労死を回避する「動的安定性」が実証された。

### **3.4. 膠着死 (Stagnation Death)**

* **観測:** ターゲット消失後、変化のない膠着状態が150ステップ継続した時点で、REBIRTH（強制再生）が発生した。
* **評価:** 機能不全に陥ったリソースを自律的に開放する循環メカニズムを確認した。

## **4. 総評**

単体ノードは、外部入力に対して「盲目的に従う」のではなく、自身の物理的慣性を保ちながら「意味のある信号」を選択的に抽出する能力を示した。この特性は、フェーズ2以降の集団（Box）において、ノイズを空間的に中和し、堅牢な共鳴経路を形成するための強固な基盤となる。
